TASK 1 OVERVIEW

This project is a command-line based To-Do List application developed using Python.
It helps users manage, organize, and track their daily tasks efficiently through a menu-driven interface.
The application allows users to add tasks, update them, mark tasks as completed, search for tasks, delete tasks, and clear all tasks.

FEATURES

Add new tasks
View all tasks with status (Completed / Pending)
Mark tasks as completed
Update existing tasks
Delete tasks
Search tasks by keyword
Clear all tasks at once
Continuous menu until user exits

TECHNOLOGIES USED

Programming Language: Python
Interface: Command Line Interface (CLI)
Editor: Visual Studio Code
