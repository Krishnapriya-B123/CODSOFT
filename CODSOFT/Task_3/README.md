PASSWORD GENERATOR (PYTHON)

TASK 3 DESCRIPTION

This project is a Password Generator application developed using Python.
It generates strong and random passwords based on the length specified by the user using a combination of letters, numbers, and special characters.

FEATURES

User-defined password length
Generates strong and random passwords
Uses uppercase letters, lowercase letters, numbers, and special characters
Simple and user-friendly interface
Fast and secure password generation

TECHNOLOGIES USED

Programming Language: Python
Libraries Used: random, string
IDE: VS Code / Any Python-supported IDE
