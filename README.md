CODSOFT PYTHON INTERNSHIP PROJECTS

PROJECT OVERVIEW
This repository contains Python projects completed as part of the CODSOFT Internship Program.
Each task demonstrates core Python concepts such as user input handling, conditional logic, loops, functions, and basic data structures.

PROJECT STRUCTURE
CODSOFT
Task_1 → TO-DO LIST APPLICATION
Task_2 → CALCULATOR
Task_3 → PASSWORD GENERATOR
Task_4 → ROCK-PAPER-SCISSORS GAME
Task_5 → CONTACT BOOK

TASK DETAILS
TASK 1 – TO-DO LIST
Add, view, update, delete tasks
Mark tasks as completed
Menu-driven interface

TASK 2 – CALCULATOR
Performs addition, subtraction, multiplication, and division
Handles division by zero
Simple user-friendly interface

TASK 3 – PASSWORD GENERATOR
Generates strong random passwords
User-defined password length
Uses letters, digits, and special characters

TASK 4 – ROCK-PAPER-SCISSORS GAME
User vs Computer gameplay
Random computer selection
Score tracking
Play-again option

TASK 5 – CONTACT BOOK
Store contact details (name, phone, email, address)
Add, view, search, update, and delete contacts
Menu-driven interface
